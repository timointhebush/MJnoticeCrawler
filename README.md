# MJnoticeCrawler

## 명지대학교 새 공지사항 크롤러
명지대학교 웹사이트에서 코로나19종합, 학사, 일반, 장학금, 취업 관련 새로운 공지('n'아이콘이 붙은)를 스크랩합니다. 원하는 경로에 해당 카테고리의 공지 제목, 글 링크를 담은 텍스트를 생성합니다. 

### 필요한 라이브러리
- ```beautifulSoup4```
